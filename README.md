    ├── discord.io@1.8.8 

    ├── imgur-node-api@0.1.0 

    ├── mathjs@3.1.4 

    ├── roll@1.1.0 

    ├── shelljs@0.6.0 

    ├── winston@2.2.0 

    ├── wordnet@0.1.2 

    └── youtube-node@1.3.0 

Most of those should already be installed in the node_modules Scratch should work almost out of the box with some config changes in config.json
